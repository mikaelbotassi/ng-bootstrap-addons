Empty Data Component

A flexible component for displaying empty states with customizable SVG icons, titles, and messages. Perfect for showing "no data available" scenarios in tables, lists, or any content area. Supports custom icons, responsive design, and Bootstrap styling.

Drag Drop Upload Component

An intuitive drag-and-drop file upload component with progress tracking and validation. Supports multiple file selection, file type restrictions, and real-time upload progress. Features a clean, responsive interface with Bootstrap styling and accessibility support.

Form Error Message Component

A smart error message component that automatically displays validation errors for form controls. Integrates seamlessly with Angular reactive forms and provides customizable error messages. Supports internationalization and follows Bootstrap form validation patterns.

Input Component

An enhanced input component with Bootstrap floating labels, password visibility toggle, and validation. Features custom SVG icons, input masking, and seamless integration with Angular forms. Supports various input types (text, email, password) and provides a consistent user experience.

Numeric Interval Input Component

An enhanced input component with Bootstrap floating labels, password visibility toggle, and validation. Features custom SVG icons, input masking, and seamless integration with Angular forms.

Autocomplete Search Component

A powerful autocomplete search component with flexible data source integration. Supports remote API calls, local filtering, custom templates, and keyboard navigation. Features debounced search, loading states, and customizable result display with Bootstrap styling.

Date Range Picker Component

A comprehensive date range picker with Bootstrap floating labels and locale support. Features separate start/end date inputs, calendar integration, and automatic validation. Supports multiple date formats, internationalization (pt-BR/en-US), and responsive design.

Switch Component

A modern toggle switch component with smooth animations and customizable styling. Supports disabled states, custom labels, and integrates perfectly with Angular reactive forms. Features Bootstrap-compatible design with accessibility support and responsive behavior.

Select Component

A feature-rich select dropdown with search functionality, custom templates, and validation support. Supports remote data loading, keyboard navigation, and Bootstrap floating labels. Features clean styling, loading states, and seamless integration with Angular forms.

Multiselect Component

An advanced multiselect component with checkboxes, search filtering, and batch operations. Features custom SVG icons, select all/none functionality, and tag-based display of selected items. Supports validation, keyboard navigation, and Bootstrap styling with responsive design.

Textarea Component

A flexible textarea component with Bootstrap floating labels, character counting, and auto-resize functionality. Features validation support, placeholder text, and seamless integration with Angular reactive forms. Supports customizable rows, maximum length limits, and responsive design.

Label Component

A customizable label component with required field indicators, tooltips, and Bootstrap integration. Features automatic asterisk display for required fields, flexible styling options, and accessibility support. Integrates seamlessly with form controls and supports internationalization.

Dynamic Size Input Component

A user-friendly dynamic size input component that adjusts its width based on the content. Features customizable min/max widths, placeholder support, and seamless integration with Angular forms. Supports Bootstrap styling, keyboard input, and responsive design for a consistent user experience.

Pagination Component

A versatile pagination component with support for various layouts, item counts, and navigation styles. Features responsive design, accessibility support, and seamless integration with Angular forms. Ideal for managing large datasets and improving user experience.

Gallery Component

A sleek gallery component for displaying images in a modal with carousel functionality. Supports image thumbnails, navigation controls, and responsive design. Features smooth transitions, keyboard navigation, and seamless integration with Angular applications.

Table Component

A powerful table component with features like sorting, filtering, and pagination. Supports customizable columns, row selection, and integrates seamlessly with Angular forms. Ideal for displaying large datasets with ease and efficiency.
